## Data Flow Example: Loading Dashboard

```
1. User visits /crm
   â†“
2. CRMDashboard.tsx calls api.crm.getDashboard()
   â†“
3. Frontend sends: GET http://localhost:8000/crm/dashboard
   â†“
4. Backend routes to: CRMController::getDashboard()
   â†“
5. Controller queries:
   - leads table (for metrics)
   - lead_activities table (for recent activities)
   - Aggregates pipeline data by stage
   â†“
6. Returns JSON:
   {
     metrics: { totalLeads, newLeads, wonDeals, ... },
     recentActivities: [...],
     pipelineData: [...]
   }
   â†“
7. Frontend receives data and renders dashboard
```
